ClickGuard API Docs

API provides several endpoints for clickbait detection and text extraction.

Extract

POST /extract

Takes in a html article and returns extracted title and main content.

Request Body:

{
    "url": "string",
    "html": "string"
}

Response:

{
    "title": "string"
    "content": "string"
}

Predict

POST /predict

Takes in an article title and content and returns a clickbait prediction based on the title and content.

Request Body:

{
    "title": "string"
    "content": "string"
}

Response:

{
    "prediction": "int",
    "probability": "float",
    "explanation": "string",
    "spoiler": "string"
}

Extract and Predict

POST /extract_and_predict

Takes in a html article, extracts title and content, and returns a clickbait prediction, probability, explanation and spoiler.

Request Body:

{
    "url": "string",
    "html": "string"
}

Response:

{
    "prediction": "int",
    "probability": "float",
    "explanation": "string",
    "spoiler": "string"
}

Predetect

POST /predetect

Takes in a html of the website, extracts articles urls and titles, and for each returns a clickbait probability.

Request Body:

{
    "url": "string",
    "html": "string"
}

Response:

{
    "predictions": {
        "url1": "int",
        "url2": "int",
        ...
    }
}
