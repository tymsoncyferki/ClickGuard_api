ClickGuard API Docs

API provides two main endpoints for clickbait prediction and text extraction from articles.

Extract

POST /extract

This endpoint takes in an article and returns extracted title and main content.

Request Body:

{
    "url": "string",
    "html": "string"
}

Response:

{
    "title": "string"
    "content": "string"
}

Errors:

{
    "error": "Validation error message"
}

Predict

POST /predict

This endpoint takes in an article and returns a clickbait prediction based on the title and content.

Request Body:

{
    "title": "string"
    "content": "string"
}

Response:

{
    "prediction": "int"
}

Errors:

{
    "error": "Validation error message"
}

Extract and Predict

POST /extract_and_predict

This endpoint takes in an article, extracts title and content, and returns a clickbait prediction.

Request Body:

{
    "url": "string",
    "html": "string"
}

Response:

{
    "prediction": "int"
}

Errors:

{
    "error": "Validation error message"
}
